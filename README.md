# UCL 2025/26 Predictor

A football analytics application combining ELO rankings, Monte Carlo simulation, 111-agent swarm intelligence and Kalshi prediction market data to forecast the UEFA Champions League 2025/26 winner.

## Stack
- Backend: FastAPI, Python
- Frontend: React, Vite
- AI: Anthropic Claude API
- Markets: Kalshi

## Setup
Copy .env.example to .env and add your keys.
See .env.example for required variables.
